KafSaxParser
version 1.0
Copyright: VU University Amsterdam
email: p.vossen@let.vu.nl


DESCRIPTION
This module parses KAF files and creates a data structure that can be accessed and modified.
It also serializes KAF to a file again.

SOURCE CODE:

https://kyoto.let.vu.nl/svn/kyoto/trunk/lib/kafsaxparser

The binaries can be built using maven and the pom.xml

> mvn install

REQUIREMENTS
KafSaxParser is developed in Java 1.6 and can run on any platform that supports Java 1.6
